# RBF

**Redis Brute Force**

![](/1.png)

- <kbd>Download / Install</kbd>

```cmd
root@kali:~# wget -q "https://raw.githubusercontent.com/d4t4s3c/RBF/main/RBF.sh" -O /usr/bin/RBF
root@kali:~# chmod +x /usr/bin/RBF
```

- <kbd>Use</kbd>

```cmd
root@kali:~# RBF -t <TARGET> -p <PORT> -w <WORDLIST>
```

- <kbd>Tested On:</kbd>

  * Kali
